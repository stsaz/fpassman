Usage:

    fpassman [OPTIONS] [-a NAME USER PASS URL NOTES] [-e ID NAME USER PASS URL NOTES] [-r ID]

Options:

 -db DBFILE         Set database file
 -compat-v1.1       Read old v1.1 DB file
 -filter STR        Set filter

 -add NAME USER PASS URL NOTES
                    Add new entry
 -edit ID NAME USER PASS URL NOTES
                    Edit existing entry
 -remove ID         Remove existing row by its ID

 -help              Print help info and exit
