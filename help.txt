Usage:
    fpassman [OPTIONS] [-a NAME USER PASS URL NOTES] [-e ID NAME USER PASS URL NOTES]

Options:
-d, --db=DBFILE   Set database file
-f, --filter=STR  Set filter

-a, --add NAME USER PASS URL NOTES
                    Add new entry
-e, --edit ID NAME USER PASS URL NOTES
                    Edit existing entry

    --compat-v1.1   Read old v1.1 DB file

-h, --help   Print help info and exit
